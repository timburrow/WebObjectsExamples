Multiple Component File to File Upload in Form (Component Action with 'streamToFilePath' Binding)

This page demonstrates the use of several WOFileUpload component using the 'streamToFilePath' binding, inside of a page with other form elements using component actions. This example is designed to take three arbitrary files and save them to disk as they are upload and then display their filepaths along with the other form values

Because the uploaded files are not displayed or read into memory, this page can be used to upload arbitrarily large file.
