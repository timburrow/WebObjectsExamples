WOComponentElements Framework

The WOComponentElements framework provides a repository for reusable components which emulate the various dynamic elements in the basic WebObjects distribution.

Using ComponentElements

If you find something useful in this framework which you'd like to use in one of your applications, simply add the WOComponentElements framework to your WebObjects Application project as a framemwork.

Interesting Items in WOExamples

Please see the WOExample example application in /Developer/Examples/JavaWebObjects for a complete discussion of the items of interest within this framework
