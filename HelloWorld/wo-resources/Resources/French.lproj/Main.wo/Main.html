Cette application est traduite en Anglais, Français, Espagnol, Japonais et Chinois. Vous pouvez changer de langue en modifiant les préférences (langues) de votre navigateur.

Cet exemple démontre la communication entre composants - et plus précisément comment utiliser le codage clés/valeurs pour passer de l'information entre objets sans avoir a implémenter les méthodes d'accés.

Dans cet exemple, nous utilisons un formulaire avec un seul champ de texte, un bouton "submit" et une méthode dans le composant Main pour passer votre nom d'une page a l'autre.
