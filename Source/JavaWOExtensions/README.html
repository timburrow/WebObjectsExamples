JavaWOExtensions

This project is provided as an example of how many common reusable components in WebObjects are implemented. You may copy and rename the source code examples provided in this project, however, you should not attempt to replace the JavaWOExtensions.framework which ships with the system by a modified version of this project. It is very likely that the implementation of the various components in this project will change from release to release, and it will be difficult to keep your modifications in sync.
